# Birthday Invitation HTML Page

Welcome to my Birthday Invitation Page project! This is a simple HTML webpage designed as an introduction to HTML and web development. It serves as a digital invitation for a birthday party.

## Features

- **Invitation Message:** A warm invitation to the birthday party.
- **Event Details:** Includes the date and a picture of a birthday cake.
- **What to Bring:** A list of items guests can bring to the party.
- **Venue Information:** A Google Maps link to the party location
